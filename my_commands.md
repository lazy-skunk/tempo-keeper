# Frequently Used Commands

## npm scripts

```bash
# App
npm run dev
npm run build
npm run start

# Quality
npm run lint
npm run format
npm run format:check

# Test
npm test
npm run test:coverage
```

## Docker

```bash
docker-compose build --no-cache
docker-compose up --build -d
docker-compose stop
docker-compose down --rmi all --volumes
```
